Getting Started

The first time you open Indesign, before opening a document, you need to set up your workspace.

  1. In your top tool bar click Indesign CC > Preferences > General
  2. Untick the option 'Show "start" Workspace When No Documents Are Open'
  3. Press 'Okay'
  4. Next, set your workspace to advanced. To do this, click Window > Workspace > Advanced
  5. Open the template you would like to edit and save a new copy. File > Save As

Introduction

to Basic Tools

Here's a quick overview of the tools you'll need to use to edit your Indesign templates:

  1. On the left of your screen is your toolbar. There are two main tools you'll need to use to edit your templates.
  2. The Selection tool (keyboard shortcut is 'v'). This tool needs to be selected in order to move any objects in the template
  3. Text tool (keyboard shortcut is 't'). This tool needs to be selected to edit text

Editing

Text

Here's how to edit existing text in your template

  1. First, click the 'Text tool' in the left toolbar or press 't' to use the keyboard shortcut
  2. Place your cursor over the text you would like to edit and click You'll notice that when you have the text tool selected and are hovering over editable text, the cursor changes to the single text cursor
  3. Type your new text as you would in any text editing software

Adding new text

  1. Using the text tool, simply click and drag to create a new text box
  2. To help maintain consistency and to aid your editing, I have created a range of text styles
  3. Look for the tab titled 'Character styles' on the right of the screen If the menu isn't expanded like the screenshot above, just click on the Character Styles title.
  4. Here you'll see you have the heirarchy of headers as well as body text. To change the size and style of text, just highlight the text you want to change and click on the correct style in the Character Styles.
  5. This is particularly useful in the recipe cards when you want to bold the ingredients - just select the ingredient you want to make bold and then choose the 'body text bold' style.
  6. If you do need to amend the font, size or line height - you can adjust this in the text toolbar at the top of the screen

Editing

Images

To simplify the edits you need to make to your templates, some elements are locked down. All the images you need to edit, should be editable. If there are any that you can't do, just email me.

  1. First, select the Selection Tool (or 'v' on your keyboard)
  2. Click on the image you would like to change, you'll see a bounding box appear
  3. To add your new image, press 'cmd + d' on your keyboard. Locate the image you would like to use using the finder window and press open Indesign works in a different way to other text editing software you may have used, instead of embedding an image into the document, it links them. If you change the location of where an image file is saved, you will get an link error warning. This is why when you've finished editing your file, you will save is as a package...but more on that later!
  4. You may need to resize your image for it to fit in the bounding box. To do this, use Indesign's resizing tools: The first tool is probably going to be the one you use the most. This will make your image fill the image frame proportionally. The next one along will fit the image proportionally to the image frame.
  5. When you have the 'Selection tool' selected, you'll see the content grabber doughnut appear when you hover over an image. If you click and drag the mouse over this doughnut, it will allow you to reposition the image within the image frame.

Re-linking missing images

If you see the error that your document contains missing image links, it is very easy to re-link them:

  1. Click on the 'Links' tab on the right tool panel
  2. Any images that are missing will have a red circle with a question mark next to it
  3. To relink, just double click on the question mark and the finder window will open. Locate where your file is saved and press 'open' Indesign is clever and if it finds the image for other missing linked images in the same folder, it will relink them.

Getting your document

Print Ready

Once you have finished editing your document, you need to ensure it is ready to be sent for printing.

  1. Click File > Export
  2. Choose 'Adobe PDF (Print)' from the Fromat drop down menu
  3. Click Save
  4. In the window that pops up, click on the tab labelled 'Marks and Bleeds' and tick the checkbox next to 'All Printer's Marks'
  5. Next, check the ckeckbox 'Use Document Bleed Settings'
  6. Finally, click on the advanced tab and in the Fonts section ensure the number reads 0%
  7. Click Export If any images are incorrectly linked, it will flag an error to ask you to check them. Don't ever send a document with missing image links to the printer.
  8. Finally, you need to ensure that your document is easy for editing in the future and all files are in the correct place. To do this you just need to click File > Package and then 'Package' again. This will create a folder containing all the document fonts, images and any other assets required.
